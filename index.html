Cómo conectar:
1. Abre script.google.com → Nuevo proyecto
2. Copia el script del campo de abajo y pégalo completo
3. Desplegar → Web App · Ejecutar como: Yo · Acceso: Cualquiera
4. Autoriza los permisos → copia la URL del Web App
5. Pégala arriba y haz clic en Probar

Hojas que se crearán/rellenarán automáticamente:
• Datos Larvicultura - M01…M10 → Calidad + PLG + Población (1 fila por TQ)
• Control_Tanque M01…M10 → Parámetros OD/°C (1 fila por TQ × hora)
• Datos Larvicultura - CIO → Igual que M01–M10 (módulo CIO)
• Control_Tanque CIO → Parámetros OD/°C del módulo CIO
• Lab_Algas → Registro diario de cultivo de algas
